# JavaScript Drum Machine

An interactive JavaScript-based Drum Machine that allows users to play different drum sounds using keyboard keys or by clicking the drum pads.

This project provides a real-time audio experience where each drum pad triggers a unique sound and displays the name of the sound being played.

## Features

The drum machine includes:

- Nine interactive drum pads
- Click-based sound triggering
- Keyboard controls using:
  - `Q`
  - `W`
  - `E`
  - `A`
  - `S`
  - `D`
  - `Z`
  - `X`
  - `C`
- Real-time display of the currently played sound
- Audio playback using HTML audio elements

## Technologies Used

- HTML5
- JavaScript (ES6)
- DOM Manipulation
- Event Listeners
- HTML Audio API

## How It Works

1. Each drum pad is connected to a unique audio clip.
2. Clicking a drum pad triggers the corresponding sound.
3. Keyboard events detect key presses and play the matching drum sound.
4. The display area updates with the name of the sound being played.

## How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Click the drum pads or use keyboard keys to play sounds.
